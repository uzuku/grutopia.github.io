class grutopia.core.datahub.IsaacData[source]¶

isaac status in grutopia

There are three types of isaac status:

  • Action

  • Observation

  • Episode

structure of isaac status like this:

{
    actions: {
        "task_0":{
            robot_1: {
                cap: param,
                controller: param,
            }
        }
    },
    observations: {
        "task_0":{
            robot_1: {
                obs_1: data,
                obs_2: data
            }
        }
    },
    task_idx_counter: 0
    finished_tasks: []
}
